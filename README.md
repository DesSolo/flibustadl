## flibustadl
Simple cli application for download books from flibusta

Simple usage:
```bash
flibustadl https://flibusta.is/a/1206
```

Features:
- Download author
- Download sequence
- Download series

Supported options:
```bash
flibustadl --help                                                                                                                                                                                                                                                                                                                                     
Usage of flibustadl:
  -books string
        Books file path (default "books/")
  -loglevel int
        Log level (default -4)
```
